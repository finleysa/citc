ROADRUNNER

Road-Runner is a multi-band fully passive GSM interception system designed to monitor the traffic between base stations and mobile phones

  • Undetectable operations in passive mode
  • Real-time 2G and GSM interception
  • Intercepted services: video, data, SMS, EMS, MMS, call forwarding
  • Target alerts can be filtered by IMSI, IMEI, MSISDN, BTS
  • Stores call metadata and identifiers in an integrated database
  • Stores all intercepted voice audio and SMS
  • Receives mixed bands (850/900/1800/1900 MHz)
  • Receives RF on 1 to 4 antennas
  • True plug-and-play external Deciphering Unit decrypts A5/1 with minimal latency
