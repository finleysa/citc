ICatch

  • Has both active and passive capabilities
  • Active mode: the IMSI Catcher mimics a cell tower and forces all nearby devices to connect to it
  • Passive mode is undetectable
  • Target alerts can be filtered by IMSI, IMEI, MSISDN, BTS
  • Simultaneous 2G and 3G capabilities (GSM900, GSM1800, UMTS900, UMTS1800)
  • IMSI, IMEI, TMSI and SIM card nationality can be extracted
  • All captured identifiers are stored indefinitely for exploitation
  • Direction finding based on timing advance signal analysis
  • Active Denial-of-Service features
  • Active forced handover (force 3G to 2G)
  • Spoof SMS and cell tower error messages
  • Track mobile phones even when SIM card is changed
  • Silent call features to extract mobile phone power usage
